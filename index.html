Python备忘笔记

Posted on 2018-06-26

类

在Python中，用
class XXX():
定义一个类
初始化方法为
def __init__(self,········):
Python中的类也有着在OOP中常见的用法

可变数量的实参

在定义函数时，可以引入一个字典来实现可变数量的实参
例如,对于下面的函数

若输入
save_info(name="lolita")
输出时就为
{'name': 'lolita'}
若输入
save_info(name="lolita", age="15")
输出时就为
{'name': 'lolita', 'age': '18'}

别名

在Python中import时，可以通过在后面加上as XXX来给引入的模块起别名

lambda(λ)

来自FP(Functional Programming)的概念

大体上来说，对于一个简单的输入x,输出x+1的函数
def fun1(x) x=x+1
可以写成
lambda x:x+1
形式的匿名函数，在一定的情况下简化代码

sorted

sorted(iterable, /, *, key=None, reverse=False)
Return a new list containing all items from the iterable in ascending order.
A custom key function can be supplied to customize the sort order, and the reverse flag can be set to request the result in descending order.
